# 04_PJT

### index
```
db에서 정보를 불러와 화면에 표시했다. 장고 문법으로 포문을 돌려 db를 나열했고 하이퍼 링크로 상세페이지로 이동하게 추가했다.
```

### new
```
디비에 저장할 정보를 적을 뉴페이지와 디비로 정보를 올리는 함수를 구분하여 작성했다. 그리고 기존 인덱스로 돌아갈 수 있게 버튼추가
```

### detail
```
index에서 디테일 버튼을 누를 때, 해당 영화의 pk 값을 같이 전송되도록 구성하고 detail 함수 내에서 db정보를 불러와 넘겨 표현했다.
```

### delete
```
delete 버튼을 눌렀을 때, 해당 상세 페이지의 pk 값을 값이 넘겨주고 db에서 값을 삭제했다. 이후 index로 리다이렉트했다.
```

### edit
```
디테일 페이지에서 pk 값을 넘겨 해당 영화의 정보를 불러와 edit 페이지에 표현했다. 이후에 update 함수를 만들어서 변경된 값을 다시 db로 올릴 수 있게 만들었다.
```
